## 🛠️ Customization

### Change Primary Color
```js
// In tailwind.config.js, change:
'neon-cyan': '#00f5ff'
// to:
'neon-cyan': '#YOUR_COLOR'
```

### Add New Animation
```css
/* In src/index.css, add: */
@keyframes newAnimation {
  0% { /* start */ }
  100% { /* end */ }
}

/* Then in tailwind.config.js: */
animation: {
  'new-animation': 'newAnimation 2s ease-in-out'
}
```

### Modify Spacing
```js
// In tailwind.config.js
spacing: {
  'custom': '1.5rem'  
}
```
